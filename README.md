This repository contains the **frontend user interface** for my full-stack Budget Management application. It was engineered to provide a clean, minimalist, and highly responsive dashboard for tracking personal finances, managing strict budgets, and visualizing cash flow.

Originally architected during my professional software development internship at CV Corp, this frontend connects to a robust Java/MySQL backend via REST APIs.

## ✨ Key Features
* **Modern UI/UX:** Built with a mobile-first approach using Tailwind CSS for fluid, responsive design across all devices.
* **Dynamic Data Visualization:** Real-time updates and interactive components for logging expenses and tracking loans.
* **API Integration:** Seamlessly communicates with a secure Java backend architecture.
* **Optimized Performance:** Leverages Next.js App Router for fast page loads and optimized rendering.
